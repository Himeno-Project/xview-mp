# XView MP 微信小程序

## 运行和配置

确保安装 node.js，然后在微信开发者工具中导入项目。

打开内建终端，输入 

```
cd miniprogram
npm install -S
```

安装依赖，然后打开工具栏的 "构建 npm" 进行构建，方可使用。

如果依赖更新，npm 也要重新构建。
